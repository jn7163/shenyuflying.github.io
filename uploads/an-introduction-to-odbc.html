An Introduction to unixODBC

An Introduction to unixODBC

Shen Yu

2017.1

what?

什么是ODBC?

ODBC -- Open Database Connectivity ODBC是MICROSOFT提出的数据库访问接口标准 ODBC规范后来被X/OPEN和ISO/IEC采纳，作为SQL标准的一部分

ODBC有什么组成

  • 应用程序（APP）
  • 驱动程序管理器（odbcad32，UnixODBC（这个也可以在windows上使用））
  • 驱动程序（postgresql ODBC Driver）
  • 数据库服务服务器 （postgresql）

why?

为什么用ODBC?

API——为应用提供一致的开发接口 应用一次开发，方便更换为不同类型的数据库

where? 如何获取?

unixODBC

  1. 官网: http://www.unixodbc.org/ 源码可以在这里下载，另外还有很多文档
  2. 在线手册:Microsoft ODBC 3.0 Programmers Reference Volume 1 and 2 https://msdn.microsoft.com/en-us/library/ms714177.aspx
  3. API reference https://msdn.microsoft.com/en-us/library/ms714562(v=vs.85).aspx
  4. EASYSOFT http://www.easysoft.com/developer/interfaces/odbc/index.html 很多关于ODBC的文章
  5. 内网wiki: http://192.168.4.28/wiki/ODBC

PostgreSQL ODBC driver

  1. https://odbc.postgresql.org/
  2. https://git.postgresql.org/gitweb/?p=psqlodbc.git

how? 如何使用

注册驱动文件

  • windows上是在注册表里添加内容
  • Linux上是在odbcinst.ini里添加内容

注册DSN

  • Windows上是通过驱动管理器去做
  • Linux上是通过odbc.ini

测试连接

  • Windows上通过界面
  • Linux上通过UnixODBC自带的isql
